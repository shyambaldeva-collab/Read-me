# ReadFlow User Flow Map ## 1. Onboarding & Upload 1. Landing Page (Value Prop) → 2. Sign Up/Login → 3. Empty Library State → 4. Upload PDF Modal → 5. Processing/Parsing Animation → 6. Book added to Library. ## 2. Core Reading Loop 1. Library Dashboard → 2. Tap Book → 3. Open Reading View at last saved position → 4. Adjust Typography/Theme → 5. Read/Scroll → 6. Highlight/Note → 7. Close App (Auto-save position). ## 3. Micro-Reading Flow 1. Dashboard/Lock Screen Widget → 2. Select "5 Min Read" → 3. App calculates chunk size → 4. Immersive 5-minute reading session → 5. Completion screen with streak update. ## 4. Insights & Analytics 1. Library → 2. Analytics Tab → 3. View Weekly Stats/Streaks → 4. Share achievements.
